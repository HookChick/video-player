使用技术： Android SDK API 19、 Vitamio框架、xUtils(网络请求视频) 、科大讯飞(集成语音说话功能)、SQLLite数据库(存储播放历史记录)

主要功能：
	1、视频播放：播放多种格式视频、播放控制、手势屏幕操作控制、监听视频网络卡顿、显示网速并显示缓冲进度、被其他软件调起
	2、视频列表：本地视频、网络视频、VIP视频列表、推荐列表(推荐算法按照类型相似度大于80%)、搜索和历史记录视频列表
	3、在线搜索：语音视频搜索功能
	4、用户操作:  登录注册、查询余额、查看购买记录、充值、付费购买视频

运行说明：
	1、此项目需要关联 vitamio 类库才能运行。 即需要把项目工程下 vitamio文件夹(vitamio官方的类库,也可自行在网上下载官方的项目工程) 和此项目目录放同一工作空间下面
	2、playerservice文件夹为web项目，作为后台服务器，使用的tomcat服务器，sql数据库脚本在该目录下的db目录下（只有vip视频付费功能才需使用，非必须依赖工程）
